### Install the bee command 


```
go install github.com/beego/bee/v2@latest

```

### Add the directory to your path 

```
export PATH=$PATH:$HOME/go/bin

```

### Create new Project

```
bee new myproject 

```
### Go project directory

```
cd myproject

```
### Download required dependencies
```
go mod tidy

```

### Run the project

```
bee run
```
